## 🚀 THE PROJECT: BIZZN.DE
- **Pitch:** Das Betriebssystem für die lokale Gastronomie (Speisekarte pflegen, Bestellungen verwalten, Zahlungen abwickeln).
- **Goal:** A modern, high-contrast, premium SaaS platform replacing paper chaos in restaurants.
- **Design DNA:** Dark Mode, neon/green accents, bold typography, crisp vector (SVG) logos.

## 💻 TECH STACK & ROADMAP
- **Web-Core:** Next.js, Tailwind CSS.
- **Backend:** Supabase.
- **Phase 1 (Current):** Web MVP.
- **Phase 2 (Roadmap):** PWA (Progressive Web App) for quick homescreen access.
- **Phase 3 (Future):** Native iOS & Android Apps built with React Native / Expo.

## 🧠 AI WORKFLOW SECRETS (Lessons Learned)
- **Visual Check Rule:** Gem 2 (Bauleiter) is NOT allowed to hand over to QA (Gem 4) before the CEO visually confirms the UI on `localhost:3000`.
- **UI/UX Delegation:** If sizing (e.g., logos) or layout proportions fail, the task is immediately pulled from Gem 2 and given to Gem 3 (Die Visionärin) for precise Tailwind patches.
- **SVG Handling:** Vector logos must be forced into the right size using strict Tailwind classes (`w-64`, etc.) and basic `width/height` props to prevent collapsing.
